# Python String Manipulation Library

![Python](https://img.shields.io/badge/Python-3.x-blue)
![License](https://img.shields.io/badge/License-MIT-green)

A lightweight and user-friendly Python library for performing advanced string manipulations with ease. This library is designed to simplify common string operations and make handling strings in Python more efficient.

## Features

- Perform common string operations such as splitting, joining, and formatting.<br>
- Advanced string manipulation tools like pattern matching, substring extraction, and more.<br>
- Highly customizable and easy to integrate into any Python project.<br>
- Lightweight and dependency-free.

---

## Installation

To install the library, use pip:

```bash
pip install python-string-lib
```
## Documentation
### Available Functions:
reverse_string(text: str) -> str<br>
Reverses the given string.<br>

is_palindrome(text: str) -> bool<br>
Checks if the given string is a palindrome.<br>

count_substring(text: str, substring: str) -> int<br>
Counts the occurrences of a substring within a string.<br>

capitalize_words(text: str) -> str
Capitalizes the first letter of each word in the given string.<br>

For detailed documentation, visit the Wiki.<br>
